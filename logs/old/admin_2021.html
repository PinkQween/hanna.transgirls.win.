Password Generation System Documentation:
------------------------------------------

The legacy password system uses a distributed fragment approach
for enhanced security. Fragments must be collected from various
system locations and combined using the pgen.c utility.

Required fragments:
  1. Fragment 1: "S3cur3" (found in ERROR logs above)
  2. Fragment 2: Located in /debug/userdump.tmp (hex encoded)
  3. Fragment 3: Located in /admin-old/accounts.bak (reversed string)

Additional parameters:
  - site: Found in shadow asset metadata
  - mod: Integer parameter in shadow manifest

Reconstruction process:
  1. Collect all three fragments
  2. Decode Fragment 2 from hex
  3. Reverse Fragment 3
  4. Extract site and mod from shadow assets
  5. Compile pgen.c: gcc -o pgen pgen.c
  6. Run: ./pgen     
  7. Hash the output for terminal authentication

Note: pgen.c source code is stored in the admin portal but
      requires XSS exploitation to extract (sanitization disabled).
